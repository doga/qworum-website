Qworum

Web applications can use the Qworum capabilities of web browsers through this class.

Methods for manipulating the Qworum sessions of browser tabs:

  • ‚ñ∂Ô∏é Qworum.eval() evaluates a Qworum script.
  • ‚ñ∂Ô∏é Qworum.setData() sets the value of a data container.
  • ‚ñ∂Ô∏é Qworum.getData() reads the value of a data container.

Methods for generating Qworum scripts that are passed as call arguments to Qworum.eval():

  • ‚ñ∂Ô∏é Qworum.Script() creates a Qworum script.
  • ‚ñ∂Ô∏é Qworum.Call(), Qworum.Goto(), Qworum.Return(), Qworum.Sequence(), Qworum.Fault(), Qworum.Try() and Qworum.Data() create instructions.
  • ‚ñ∂Ô∏é Qworum.Json() and Qworum.SemanticData() create data values.

Other methods:

  • ‚ñ∂Ô∏é Qworum.checkAvailability() verifies that a website can use the Qworum capabilities of browsers.

Note: The üìù sign indicates a function that is used for generating Qworum scripts, and the üöÄ sign is for functions that call the Qworum browser extension.
